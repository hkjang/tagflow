# 관리자 시작하기

TagFlow RFID 태그 관리 시스템의 관리자 계정으로 시작하는 방법을 안내합니다.

## 첫 로그인

### 1. 애플리케이션 실행

TagFlow 애플리케이션을 실행하면 로그인 화면이 표시됩니다.

### 2. 기본 관리자 계정으로 로그인

최초 설치 후 다음 기본 계정으로 로그인할 수 있습니다:

- **사용자명**: `admin`
- **비밀번호**: `admin123`
- **역할**: 관리자 (Admin)

```
⚠️ 보안 경고
기본 비밀번호는 반드시 첫 로그인 후 즉시 변경하세요!
```

### 3. 로그인 화면

1. 사용자명 입력란에 `admin` 입력
2. 비밀번호 입력란에 `admin123` 입력
3. "로그인" 버튼 클릭

로그인에 성공하면 대시보드로 이동합니다.

## 대시보드 개요

관리자 대시보드는 시스템 전체를 한눈에 볼 수 있는 중앙 화면입니다.

### 주요 영역

#### 1. 상단 네비게이션 바

- 사용자 정보 표시
- 로그아웃 버튼
- 현재 시간

#### 2. 사이드바 메뉴

관리자는 다음 메뉴에 접근할 수 있습니다:

- **대시보드** - 홈 화면
- **태그 이벤트** - RFID 태그 스캔 이벤트 조회
- **리포트** - 통계 및 성능 지표
- **관리** (Admin 전용)
  - 사용자 관리
  - 웹훅 설정
  - 정리 로그
  - 시스템 설정

#### 3. 메인 콘텐츠 영역

선택한 메뉴에 따라 다양한 정보와 기능이 표시됩니다.

### 대시보드 위젯

관리자 대시보드에는 다음 정보가 표시됩니다:

- **오늘 처리된 이벤트 수**: 금일 수신된 RFID 태그 이벤트 총 개수
- **활성 웹훅**: 현재 활성화된 웹훅 개수
- **등록된 사용자**: 시스템에 등록된 전체 사용자 수
- **최근 이벤트**: 가장 최근에 수신된 태그 이벤트 목록
- **웹훅 성공률**: 웹훅 전송 성공률 그래프

## 네비게이션

### 메뉴 이동

1. **사이드바 메뉴 사용**

   - 왼쪽 사이드바에서 원하는 메뉴를 클릭
   - 선택된 메뉴는 하이라이트 표시

2. **빠른 이동**
   - 대시보드의 위젯 카드를 클릭하면 해당 상세 페이지로 이동
   - 예: "활성 웹훅" 클릭 → 웹훅 관리 페이지로 이동

### 주요 화면 구성

#### 태그 이벤트 화면

```
/events
```

- 모든 RFID 태그 스캔 이벤트를 시간순으로 표시
- 필터링 및 검색 기능
- 실시간 업데이트

#### 리포트 화면

```
/reports
```

- 이벤트 통계
- 웹훅 성능 지표
- 시간대별 분석

#### 사용자 관리 화면 (Admin 전용)

```
/admin/users
```

- 사용자 목록 조회
- 신규 사용자 생성
- 사용자 편집/삭제
- 역할 변경

#### 웹훅 관리 화면 (Admin 전용)

```
/admin/webhooks
```

- 웹훅 설정 관리
- 필드 매핑 구성
- 웹훅 테스트
- 웹훅 로그 조회

## 첫 사용 체크리스트

관리자로 처음 시스템을 설정할 때 다음 단계를 수행하세요:

- [ ] **1. 기본 비밀번호 변경**
  - 사용자 관리 → 'admin' 사용자 편집 → 비밀번호 변경
- [ ] **2. 운영자 계정 생성**
  - [사용자 관리 가이드](user-management.md) 참조
  - 운영자 역할로 새 사용자 생성
- [ ] **3. 웹훅 설정** (필요한 경우)
  - [웹훅 설정 가이드](webhook-configuration.md) 참조
  - 외부 시스템과 연동할 웹훅 구성
- [ ] **4. 데이터 보존 정책 확인**
  - [데이터 정리 가이드](data-cleanup.md) 참조
  - 자동 정리 정책 이해
- [ ] **5. RFID 리더 테스트**
  - 태그 이벤트 페이지에서 실제 태그 스캔 테스트
  - 이벤트가 정상적으로 기록되는지 확인

## 자동 데이터 정리

```
⚠️ 중요
관리자가 로그인하면 자동으로 데이터 정리 작업이 실행됩니다.
```

로그인 시 다음 작업이 자동 수행됩니다:

1. **태그 이벤트 정리**: 1년 이상 된 이벤트 삭제
2. **웹훅 로그 정리**: 90일 이상 된 로그 삭제
3. **정리 로그 정리**: 2년 이상 된 정리 로그 삭제

정리 작업 결과는 "정리 로그" 메뉴에서 확인할 수 있습니다.

## 권한 이해

관리자 역할은 시스템의 모든 기능에 접근할 수 있습니다:

### ✅ 관리자 전용 권한

- 사용자 생성/편집/삭제
- 웹훅 생성/편집/삭제
- 웹훅 테스트 실행
- 정리 로그 조회
- 시스템 설정 변경
- 수동 데이터 정리 실행

### ✅ 모든 역할 공통 권한

- 태그 이벤트 조회
- 리포트 조회
- 웹훅 로그 조회 (읽기 전용)
- 자신의 비밀번호 변경

## 보안 권장사항

### 비밀번호 정책

- 최소 8자 이상
- 영문, 숫자, 특수문자 혼합 권장
- 주기적으로 변경 (3개월마다 권장)
- 기본 비밀번호는 절대 사용 금지

### 계정 관리

- 필요한 사용자만 생성
- 퇴사자 계정은 즉시 삭제
- 운영자 역할은 필요한 사람에게만 부여
- 관리자 역할은 최소한으로 유지

### 웹훅 보안

- HTTPS URL만 사용
- 웹훅 대상 URL을 정기적으로 검토
- 민감한 데이터는 필드 매핑에서 제외

## 문제 해결

### 로그인할 수 없어요

**증상**: 기본 계정으로 로그인이 안 됩니다.

**해결책**:

1. 사용자명과 비밀번호를 정확히 입력했는지 확인
2. 대소문자 구분에 주의
3. 데이터베이스가 초기화되었는지 확인 (`npm run seed` 실행)

### 대시보드가 비어있어요

**증상**: 대시보드에 데이터가 표시되지 않습니다.

**해결책**:

1. 아직 태그 이벤트가 없는 경우 정상입니다
2. 태그 이벤트 페이지에서 테스트 이벤트 생성
3. 백엔드 서버가 정상 작동하는지 확인

### 메뉴가 보이지 않아요

**증상**: 관리 메뉴가 보이지 않습니다.

**해결책**:

1. 관리자 계정으로 로그인했는지 확인
2. 운영자 계정은 관리 메뉴에 접근할 수 없습니다
3. 페이지를 새로고침하여 세션 재확인
